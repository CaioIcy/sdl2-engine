Contains all the necessary functionalities to loop and update the game.

sdl2engine::Game::Game ( const ConstructorTag &  private_tag)
explicit

The constructor.

Since the std::make_unique in Game::Instance() can't access a private constructor, we make this constructor public but need a ConstructorTag parameter. ConstructorTag is an empty struct only accessible by this class, since it is private. This way the constructor can only actually be used in a private setting.
